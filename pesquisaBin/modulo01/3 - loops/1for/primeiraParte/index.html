O Conhecimento de Wakanda (A Anatomia do Loop for)

O loop for é como a linha de montagem das Indústrias Stark. Ele tem três partes essenciais dentro dos parênteses ( ), separadas por ponto e vírgula ;:

for ( inicialização ; condição ; incremento ) { ... }

Inicialização (let i = 1): A criação de uma variável contadora. É o "comece a contar a partir do Dróide nº 1". (i é uma convenção para "índice" ou "iterador").

Condição (i <= 10): O critério de parada. O loop vai continuar ENQUANTO esta condição for verdadeira. "Continue montando dróides enquanto o número do dróide for menor ou igual a 10."

Incremento (i++): O que acontece no final de cada repetição. i++ é um atalho para i = i + 1. "Depois de montar um dróide, passe para o próximo número."




Seu Desafio do Disparo Repulsor

Tony Stark está calibrando os raios repulsores da sua nova armadura. Ele precisa disparar 10 vezes para coletar dados.

No final do seu arquivo, escreva um loop for.

O loop deve começar em 1 e ir até 10.

Dentro do loop, imprima no console uma mensagem para cada disparo. Por exemplo:
"Disparando raio repulsor nº 1!", "Disparando raio repulsor nº 2!", ... até o 10.
