Switch case

Passo 2: O Conhecimento de Wakanda (O Conceito de switch)

Pense no switch como o Doutor Estranho usando o Olho de Agamotto para ver futuros específicos.

switch (variavel): Você diz ao Olho de Agamotto em qual "ponto do tempo" (variável) ele deve focar.

case "valor_exato":: Cada case é um futuro possível. "No caso da variável ser EXATAMENTE igual a 'valor_exato', faça isso."

break;: Esta é a parte MAIS IMPORTANTE. O break diz ao Doutor Estranho: "Encontramos o futuro certo, pode parar de procurar!". Se você esquecer o break, o código continua executando os cases abaixo, mesmo que não correspondam, causando o caos no multiverso.

default:: Este é o "plano B" do Doutor Estranho. Se nenhum dos cases corresponder ao valor da variável, o código dentro do default será executado. É o else do mundo switch.

Seu Desafio da Joia do Infinito
Agora é sua vez. Você tem a manopla, mas precisa saber qual poder ela te dá com base na cor da Joia.

Crie uma variável const corDaJoia = "Vermelha";.

Escreva uma estrutura switch que verifique a variável corDaJoia.

Crie um case para as seguintes cores:
case "Vermelha": deve imprimir "Poder: Realidade"
case "Azul": deve imprimir "Poder: Espaço"
case "Amarela": deve imprimir "Poder: Mente"
case "Verde": deve imprimir "Poder: Tempo"
case "Roxa": deve imprimir "Poder: Poder"

Não se esqueça do break em cada case!

Adicione um default que imprima "Joia desconhecida! Cuidado, pode ser perigoso."
